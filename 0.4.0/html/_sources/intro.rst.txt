Introduction
============

``libpypsg`` is a Python wrapper for the Planetary Spectrum Generator (PSG)

Installation
************

To install using pip:

.. code-block:: shell

    pip install libpypsg

or in development mode:

.. code-block:: shell

    git clone https://github.com/tedjohnson12/libpypsg.git
    cd libpypsg
    pip install -e .[dev]
