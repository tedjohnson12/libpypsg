GCMdecoder#

class libpypsg.globes.GCMdecoder(header: str, dat: bytes)#

Bases: object

A container for a PSG GCM binary.

This class can read a PSG config containing a GCM and parse the bytes sequence.

Parameters:

  • header (str) – The header of the GCM.

  • dat (bytes) – The binary data of the GCM.
