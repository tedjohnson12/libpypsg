class libpypsg.cfg.base.Field(name: str, default: Any = None, null: bool = True)#

Bases: ABC

A data field for storing PSG configurations.

Parameters:

  • name (str) – The name of the field.

  • default (any, optional) – The default value of the field. Default is None.

  • null (bool, optional) – If false, the field cannot be empty. Default is False.
