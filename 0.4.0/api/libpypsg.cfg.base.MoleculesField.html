MoleculesField#

class libpypsg.cfg.base.MoleculesField(default: Any = None, null: bool = True)#

Bases: Field

A Field to store molecular data.

Parameters:

  • default (Any, optional) – The default value of the field. Default is None.

  • null (bool, optional) – If false, the field cannot be empty. Default is True.
