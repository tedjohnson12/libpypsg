PyGCM#

class libpypsg.globes.PyGCM(pressure: Pressure, temperature: Temperature, *args: Molecule | Aerosol | AerosolSize | Surface, wind_u: Wind = None, wind_v: Wind = None, tsurf: SurfaceTemperature = None, psurf: SurfacePressure = None, albedo: Albedo = None, emissivity: Emissivity = None, lon_start: float = -180.0, lat_start: float = -90.0, desc: str = None)#

Bases: object

A Python container for GCM data.

Parameters:

  • pressure (libpypsg.globes.structure.Pressure) – The pressure variable.

  • temperature (libpypsg.globes.structure.Temperature) – The temperature variable.

  • wind_u (libpypsg.globes.structure.Wind, optional) – The west-to-east wind variable.

  • wind_v (libpypsg.globes.structure.Wind, optional) – The south-to-north wind variable.

  • tsurf (libpypsg.globes.structure.SurfaceTemperature, optional) – The surface temperature variable.

  • psurf (libpypsg.globes.structure.SurfacePressure, optional) – The surface pressure variable.

  • albedo (libpypsg.globes.structure.Albedo, optional) – The albedo variable.

  • emissivity (libpypsg.globes.structure.Emissivity, optional) – The emissivity variable.

  • *args (libpypsg.globes.structure.Molecule | libpypsg.globes.structure.Aerosol | libpypsg.globes.structure.AerosolSize | libpypsg.globes.structure.Surface) – Additional variables.
